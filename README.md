# Emotion Brooch
<img src="image/image_screen.png"/>

嗨嗨嗨 ~ 这里是一群年轻的小伙伴们

- 项目起源于一场黑客马拉松队员们的头脑风暴，在一次次的灵感迸发中诞生了本项目的雏形，最终带着Demo杀入决赛；
- 虽然最终并未获奖，但我们相信这个有趣的项目并不只能止步于此，于是我们重新思考，推翻重来，便诞生了这个项目。
- 目前该项目还在招兵买马阶段，欢迎任何感兴趣的小伙伴加入
- 联系方式：priest_cr@163.com
